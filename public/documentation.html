Documentation for developers

1) Users will be able to create an account
2)Users will be able to update the account (name, age, and other personal info.)
3) User will be able to create a list of favorite movies.

Request URL Response
Get a list of all Movies /movies/ GET
Get movie by title /movies/:title/ GET
Allow new user to register /user/ POST
Allow user to Update BIO /user/:name PUT
Allow user to Create Favorites /user/:name/movie/movie:title POST
Allow user to remove from Favorites /user/:name/movie/movie:title DELETE
Allow User to delete account /user/:name DELETE
